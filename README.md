# 🔐 ChainGuard: Multi-Factor Authentication with Blockchain

## 🚀 Overview

A secure authentication system that combines multiple verification factors with blockchain to protect user identity and financial transactions.

## 🛠️ Tech Stack

* Python
* Node.js
* Express.js
* MongoDB
* Blockchain (custom implementation)
* JavaScript / HTML / CSS

## ✨ Key Features

* 🔑 Multi-factor authentication using PIN + image-based verification
* 🖼️ Image-based login (select 3 images in correct order from 9)
* 🔗 Blockchain-secured storage of user credentials and transactions

## ⚙️ How to Run

1. Clone the repository

   ```bash
   git clone https://github.com/Swathy8103/MFA-Blockchain-Project.git
   cd MFA-Blockchain-Project
   ```
2. Install dependencies

   ```bash
   npm install
   ```
3. Start the server

   ```bash
   node server.js
   ```
4. Open in browser

   ```
   http://localhost:3000
   ```
